# PluginList

The plugin has last been updated to function with **[Dalamud 15.0.0](https://github.com/goatcorp/Dalamud)** under FFXIV's **[7.5](https://na.finalfantasyxiv.com/lodestone/topics/detail/07320affa7e0fcd9685afcbe54fbf55405b6d822)** Patch.

## _"Your favorite tools, now just one hover away."_

Do you find yourself opening the same plugins over and over, every single session? **PluginList** gives you a clean edge dock where your most-used tools are always ready, so that you can quickly open any plugin's config menu with a mere click instead of a command!

Pin your plugins, custom commands, and in-game macros once, then just hover the screen edge and click what you need to use right away.

## Commands

- **/pluginlist**: Opens the settings window.
- **/pl**: Alias for `/pluginlist`.

## How to Use

- Import the above repository into Dalamud Settings by going into **Experimental** > **Custom Plugin Repositories**.
- Look for **PluginList** in the Dalamud Plugin Installer.
- Install the plugin and open its settings with **/pluginlist**.
- Pick your shortcuts from the **Plugins**, **Commands**, and **Macros** tabs.
- Organize everything in **Organization** (Alphabetical or Manual Drag & Drop).
- Hover your selected screen edge and use things instantly.

## Main Categories

PluginList is split into simple categories so setup stays clean and fast:

| Category         | Description                                                                                                                                       |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| **Plugins**      | Shows all of your loaded plugins and lets you pin/unpin them from the hover dock.                                                                 |
| **Commands**     | Lets you create named custom command shortcuts. Supports single-line or multi-line command text.                                                  |
| **Macros**       | Lets you pin in-game macros (Individual and Shared) directly into the hover dock.                                                                 |
| **Organization** | Lets you sort shortcuts alphabetically or manually via drag & drop, move items between Main and Unpinned, and assign custom shortcut text colors. |

## Hover Dock Behavior

The hover dock is designed to stay out of the way until you need it:

| Feature                  | Description                                                        |
|--------------------------|--------------------------------------------------------------------|
| **Edge Placement**       | Place the dock on Top, Bottom, Left, or Right.                     |
| **Edge Offset Dragging** | Drag the dock header area to move it along the selected edge.      |
| **Auto Expand/Collapse** | The dock expands while hovered and collapses when you move away.   |
| **Quick Settings Access**| Click the gear icon inside the dock to open PluginList's settings. |

## Shortcut Interaction

| Shortcut Type   | Interaction                                                                                              |
|-----------------|----------------------------------------------------------------------------------------------------------|
| **Plugin**      | Left-click opens plugin main UI (or config if no main UI). Right-click opens config UI when available.   |
| **Command**     | Left-click executes the saved command text.                                                              |
| **Macro**       | Left-click executes the selected in-game macro.                                                          |
